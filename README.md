# PhysX Toy

PhysX Toy 是一个基于 PhysX 的物理引擎的小玩具，用于学习物理引擎的基本使用

## 环境配置

### 步骤 1：下载并安装 vcpkg

在命令行中执行以下命令：

```bash
git clone https://github.com/microsoft/vcpkg.git
```

### 步骤 2：安装 glm、glew、freeglut、physx

在命令行中执行以下命令：

```bash
vcpkg install glm
vcpkg install glew
vcpkg install freeglut
vcpkg install physx
```
